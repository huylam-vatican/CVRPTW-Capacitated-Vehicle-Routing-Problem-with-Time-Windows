# CVRPTW - Vehicle Routing Optimization

Solving the Capacitated Vehicle Routing Problem with Time Windows (CVRPTW) using IBM CPLEX.

Optimizes delivery routes from a depot to 17 supermarket locations with capacity and time window constraints.

## How to Run

1. Open IBM ILOG CPLEX Optimization Studio
2. Create a new OPL Project
3. Import `CVRPTW.mod` and `CVRPTW.dat`
4. Place `CVRPTW_ProjectData.xlsx` in the same directory as `.dat` file
5. Create a Run Configuration and execute
